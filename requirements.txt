### Aside: Windows-Specific Dependencies
These are separated as they are platform-specific to Windows environments (e.g., via MSVC or MinGW). They are not part of the core POSIX/C standard list above.
direct.h - Windows Standard (part of MSVC or MinGW; use Windows SDK 10.0.26100)
windows.h - Windows SDK 10.0.26100 (for Windows 11 24H2, released in 2025)
dlfcn.h (Windows variant) - dlfcn-win32 1.4.2 (last updated in May 2025)
